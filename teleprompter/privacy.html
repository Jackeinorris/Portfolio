Política de Privacidade — Teleprompter JVDias

Última atualização: 8 de julho de 2026

O Teleprompter JVDias é um aplicativo para Windows que funciona offline e não coleta, armazena, vende ou compartilha dados pessoais.

Roteiros e preferências

Os roteiros, configurações e dados de autosave ficam armazenados localmente no computador do usuário. Nenhum roteiro é enviado para servidores externos do desenvolvedor.

Conta, anúncios e analytics

O Teleprompter JVDias não exige login, não possui anúncios e não utiliza analytics.

Controle remoto pelo celular

Quando o usuário ativa o controle pelo celular, o app abre uma comunicação local entre o computador e o telefone na mesma rede Wi-Fi. Essa comunicação serve para controlar funções como play/pause, velocidade e navegação do roteiro. O tráfego fica na rede local do usuário e não é enviado para servidor externo do desenvolvedor. Um PIN protege o pareamento, e o app bloqueia temporariamente novas tentativas após vários PINs incorretos.

Relatórios de erro

Em caso de falha, o app pode gerar um registro técnico local no computador do usuário. Esse arquivo só é enviado ao desenvolvedor se o próprio usuário decidir compartilhá-lo para suporte.

Compartilhamento de dados

Como o Teleprompter JVDias não coleta dados pessoais, não há venda, compartilhamento ou transferência de dados pessoais para terceiros.
